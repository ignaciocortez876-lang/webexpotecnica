Proyecto de Algoritmos y Programación y de Automatización

Escuela Industrial N°9 | Expotécnica 2025

1. El Problema y la Solución Automatizada

El desafío de la agricultura moderna es maximizar la producción minimizando el uso de recursos.

Nuesto proyecto, Ind N°9, presenta un prototipo funcional de un invernadero automatizado diseñado para simular y controlar el microclima y el proceso de siembra de manera autónoma, aplicando los conocimientos adquiridos en robótica y lógica de programación.

Este modelo sienta las bases para un sistema escalable, eficiente y preciso.

2. Arquitectura y Componentes del Prototipo

Diagrama de la maqueta del invernadero:

Actuadores (Output)

  • Motor de Labranza: Este motor fue reutilizado de una impresora en desuso, su funcion principal en este proyecto es la de arar las semillas que sean sembradas. Atraves de pulsadores este motor se mueve de izquierda a derecha logrando asi abrir la tierra y hacer espacio para las semillas cuando va hacia la izquierda y cubrirlas de tierra cuando vuelve hacia a la derecha.
  • Servomotor Dispensador de Semillas: Este servomotor cumple la funcion de inclinar la pequeña caja la cual contiene las semillas a sembrar. Este una vez accionado activa su brazo lo cual hace que la caja contenedora deje caer las semillas luego de que el motor de labranza abra la tierra y les haga espacio.
  • Cooler (Ventilador): Este ventilador cumple la funcion de refirgerar la zona de cultivo gracias al aire fresco que ingresa atraves del mismo
  • Bomba de Riego: Esta bomba de agua se encarga de suministrar el liquido suficiente para la tierra de los cultivos a traves de mangueras. Esta bomba de agua es suministrada gracias a las mangueras y un recipiente plastico lleno de agua.

3. Lógica de Control y Algoritmos (Teoría y Taller)

La lógica de control del invernadero se basa en la lectura de pulsadores que simulan las decisiones de un sistema automatizado. Cada entrada activa una salida específica (motor, servo, cooler o bomba). Esto se logra utilizando estructuras condicionales if y tiempos controlados con delay().

Ejemplo de Algoritmo de Riego

La bomba de riego se activa al presionar un pulsador. Esto simula el funcionamiento automático que podría implementarse con un sensor de humedad del suelo. Al activarse el relé, la bomba se enciende por un tiempo determinado y luego se apaga automáticamente.

Control de Movimiento Lineal

El motor de labranza se controla con dos pulsadores: uno para el movimiento hacia la izquierda y otro para la derecha. Este control se logra con dos señales digitales (IN1 e IN2) que determinan la dirección del giro.

4. Del aula al campo: Visión de Escalabilidad Industrial

Nuestro prototipo fue diseñado con un enfoque modular, pensado para que cada maqueta funcione como una zona de cultivo inteligente dentro de una estructura mayor, como un galpón agrícola. De esta forma, la maqueta no solo representa un modelo didáctico, sino una unidad real de automatización adaptable a diferentes escalas.

Escenario de Escalado: Galpón de Producción

En una aplicación real, el sistema podría instalarse dentro de un galpón agrícola estándar de aproximadamente 30 metros de largo por 10 metros de ancho y 4 metros de alto (unos 300 m²). Este espacio se dividiría en módulos independientes equivalentes a nuestra maqueta. Cada módulo controlaría su propio riego, ventilación y siembra, permitiendo que el sistema funcione incluso si una zona presenta fallas o condiciones diferentes.

Modelo MLC-CC (Módulo Local de Control - Controlador Central)

  • Módulo Local de Control (MLC): Representado por nuestra maqueta actual. Es un microcontrolador (por ejemplo, Arduino o ESP32) que gestiona los sensores y actuadores de una sola zona. Toma decisiones inmediatas (activar el ventilador, regar, arar o sembrar).
  • Controlador Central (CC): En un sistema a gran escala, un CC (por ejemplo, una Raspberry Pi o servidor central) recibiría datos de todos los MLCs mediante una red inalámbrica (WiFi o LoRa), analizando el estado general del galpón y enviando órdenes coordinadas a cada zona.

Materiales y Adaptaciones a Gran Escala

Para construir el sistema real, los materiales serían reemplazados por versiones industriales:

  • Estructura del galpón: Tubos de acero galvanizado o aluminio.
  • Rieles y soportes: Metales livianos o acero inoxidable.
  • Paredes y techo: Policarbonato alveolar o film agrícola traslúcido.
  • Sistema de riego: Tuberías de PVC, válvulas solenoides y control automatizado.
  • Ventiladores: Coolers industriales de 12V o 220V controlados por relés.

Funcionamiento Coordinado

Cada módulo del galpón opera de manera individual pero está conectado al sistema central. Esto permite que el invernadero responda de forma inteligente:

  • Si una zona detecta temperatura alta, activa solo su ventilador.
  • Si el suelo está seco, el riego se enciende en esa zona sin afectar las demás.
  • Si se programa una siembra general, los servos de cada módulo actúan en secuencia.

Ventajas del Sistema Escalable

  • Modularidad: cada módulo funciona de forma independiente.
  • Ahorro de agua y energía: solo se activan los sistemas necesarios.
  • Fácil mantenimiento y expansión.
  • Monitoreo centralizado con posibilidad de app o interfaz web.

En resumen, cada maqueta representa una célula de automatización dentro de un sistema agrícola mayor. Al replicar decenas de estas unidades dentro de un galpón, se obtiene un invernadero industrial inteligente, eficiente y adaptable a distintas condiciones de cultivo.
